1) Нажмите/введите start, чтобы начать.

Доступные комманды:
/createcustom - конструктор опроса
/createjson - конструктор опроса в формате Json
/createtext - конструктор опроса в формате txt
/join [tokenID] - получение доступа к опросу по ключевому слову.
/getstats [tokenID] ([pass]) - получение статистики по опросу (закрытому)
/showpublicsurveys - список открытых опросов, доступных для прохождения

2) *ввод команды*

-/createcustom
->переход в конструктор опросов
-> 4)

-/createjson
->переход в конструктор json опросов
-> 4)

-/createtext
->переход в конструктор txt опросов
-> 4)

-/getstats [tokenID] ([pass])
->вывод статистики по опросу ID, если пароль введен правильно (если опрос требует пароль)
-> 2)

-/showpublicsurveys
->вывод списка открытых к прохождению опросов
-> 2)

-/join [tokenID]
->переход к опросу номер ID 
-> 3.1) or 3.2)


3.1) Хотите получить статистику? (доступ есть)
(да/нет)
3.1.1) Вывод статистики

3.2.1) (кс) Вывод закрытой статистики
3.2.1) -> 2)
